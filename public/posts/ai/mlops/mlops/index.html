MLOps - Skills Required

March 9, 2024

Here are a list of skillsrequired for a career in MLOPs?

  1. Programming:
    1. Python
    2. Tools (IDEs, testing, build tools, CI/CD)
    3. Libraries (Langchain, Llamaindex)
    4. ML Chatbots
    5. Data Engineering (pandas)
  2. Containers and Kubernetes
  3. ML Fundamentals
    1. Deep Learning
    2. Foundation Model - Architectures
      1. LLM
      2. Vision models
      3. MMLU
    3. Prompt Engineering
    4. Embedding, Vector Stores
    5. RAG
  4. MLOps
    1. MLOps Principles
    2. MLOps Components
      1. Version Control & CI/CD
      2. Orchestration
      3. Experminet tracking (MLflow)
      4. Data lineage
      5. Model training & serving
      6. Monitoring and Observability
    3. Infrasrucutre as Code
      1. Terraform
      2. BLIP
      3. CloudFormation
